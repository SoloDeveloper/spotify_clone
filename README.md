##  Spotify Clone
Developed a full stack Spotify clone to continue my learning with Software Engineering.

## Features
User authentication
Music playback and controls
Library management
Search functionality
Payment processing with Stripe

## Technologies Used
Next.js
React
Stripe
Supabase
Tailwind CSS
